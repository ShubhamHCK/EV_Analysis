📘 About This Project

This project focuses on analyzing Electric Vehicle (EV) data using Tableau dashboards and integrating them into a Flask-based web application. It provides insights into charging patterns, battery performance, and efficiency across different EV models.

⚡ Data Analysis

Analyzing EV datasets for patterns and insights

📊 Visualization

Interactive dashboards created using Tableau

🌐 Web Integration

Flask-based web app embedding dashboards

🎯 Objectives

  • Understand EV charging behavior
  • Analyze battery efficiency and range
  • Compare EV models based on performance
  • Provide data-driven insights
